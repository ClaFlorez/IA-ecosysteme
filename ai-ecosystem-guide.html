🎯 Définition du métier Machine Learning Engineer

  • Missions principales: Concevoir, développer et déployer des modèles ML en production
  • Livrables types: Pipelines ML, modèles entraînés, APIs de prédiction, documentation technique
  • Mots-clés techniques: Python, TensorFlow, PyTorch, MLOps, Kubernetes, Docker
  • Alias de recherche: ML Engineer, AI Engineer, Applied ML Scientist

💰 Niveaux de salaire (données disponibles)

  • Montréal: 100K - 240K CAD
  • Toronto: 130K - 170K CAD
  • New York: 147K - 275K USD
  • Canada (moyenne): 142K CAD/an
